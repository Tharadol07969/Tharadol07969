- **🏆 Achievements:**

  - Improved downstream analytics efficiency by ensuring consistent data quality and schema integrity.
- **🎯 Technologies used:**
    - **Database & Querying:** PostgreSQL (v17), pgAdmin 4  
    - **Data Processing:** SQL (CTEs, window functions, PERCENTILE_CONT)
